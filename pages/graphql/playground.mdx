## Intuitive Playground Editor
The Playground Editor in Firecamp offers a user-friendly interface to help you write and prepare GraphQL operations quickly. Whether you are creating queries, mutations, or subscriptions, the Playground Editor provides features such as **syntax highlighting**, **autocompletion**, and **error handling**, making it easy to build and test your GraphQL operations efficiently.

### Multi-tabs Playgrounds
For GraphQL request in Firecamp, you have the flexibility to open multiple playgrounds within the same GraphQL request. This feature allows you to work on different GraphQL operations simultaneously, making it convenient to test and compare different queries, mutations, or subscriptions within a single session. By utilizing multiple playgrounds, you can efficiently manage and organize your GraphQL workflow without any limitations.
